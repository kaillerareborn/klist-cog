# klist
Cog for [Red-DiscordBot](https://github.com/Cog-Creators/Red-DiscordBot).

Set the channel ids where stated in the code. The cog starts as soon as you load it.

It checks the servers and waiting games on [Kaillera Reborn](https://kaillerareborn.github.io)'s lists and posts it to Discord, then it gets updated once a minute.
